# KJTheRealtor

Branded multi-page real estate website for Kareem Jamal of Rodeo Realty Fine Estates.

## Live Pages

- `/` - Homepage
- `/buyers-and-sellers` - Buyer representation and listing services
- `/family-wealth-preservation` - Long-term real estate and family wealth planning
- `/mls-search` - CRMLS IDX search experience

## Project Notes

- Built as a static HTML/CSS/JS site for fast deployment on Vercel.
- Uses clean URLs via `vercel.json`.
- Uses Formspree for lead capture.
- Uses a CRMLS frameable IDX embed for Phase 1 MLS search.
